# Shivansh Restaurant Website 🍽️

A basic restaurant website created using **HTML and CSS**.

This project was built as a beginner frontend practice project to improve my understanding of:

- HTML structure
- CSS styling
- Navigation bars
- Images
- Forms
- Multiple webpages
- Linking pages together

## Features

✅ Home page  
✅ Menu page  
✅ Price section  
✅ Online order page  
✅ Food images  
✅ Basic restaurant UI  
✅ Order form  

## Technologies Used

- HTML5
- CSS3

## Project Structure

```bash
restaurant-project/
│
├── index.html
├── menu.html
├── price.html
├── order.html
├── style.css
├── images/
```

## How to Run

1. Download the project
2. Open the project folder
3. Double click `index.html`

OR

Open using VS Code Live Server.
